# Introduction

Welcome to **a-Library** – your comprehensive technical documentation library built with [Astro Starlight](https://starlight.astro.build/).

## What is a-Library?

a-Library is a knowledge base designed to help developers quickly find accurate, up-to-date documentation. It brings together guides, references, and tutorials in a single, searchable location.

## Key Features

- 🔍 **Full-text search** – Find any content instantly using the search bar (press <kbd>/</kbd> or <kbd>Ctrl+K</kbd> to open search).
- 📚 **Comprehensive guides** – Step-by-step documentation for common workflows.
- 🗂️ **Organized references** – Detailed API and configuration references.
- 🎓 **Tutorials** – Practical, hands-on tutorials for building real applications.

## How to Use This Library

Use the **sidebar** on the left to browse documentation by category, or use the **search box** at the top of the page to find exactly what you need.

> **Tip:** Press <kbd>/</kbd> or <kbd>Ctrl+K</kbd> anywhere on the site to open the search dialog.
