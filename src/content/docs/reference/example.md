## Reference Content

Reference pages are ideal for presenting technical specifications, APIs, and configuration options.

### Configuration Options

| Option      | Type     | Description                               |
|-------------|----------|-------------------------------------------|
| `title`     | `string` | The title of the documentation page.     |
| `description` | `string` | A brief description of the page content. |
| `template`  | `string` | The page template (`doc` or `splash`).   |
